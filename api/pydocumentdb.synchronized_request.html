pydocumentdb.synchronized_request module¶

Synchronized request.

pydocumentdb.synchronized_request.SynchronizedRequest(connection_policy, method, base_url, path, request_data, query_params, headers)¶

Performs one synchronized http request according to the parameters.

Parameters:
  • connection_policy: documents.ConnectionPolicy
  • method: str
  • base_url: str
  • path: str
  • request_data: str, unicode, file-like stream object, dict, list or None
  • query_params: dict
  • headers: dict
Returns:

tuple of (result, headers), where both result and headers are dicts.
